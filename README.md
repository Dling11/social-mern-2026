# Social Platform

Full-stack MERN social media application built incrementally with a scalable React + Express architecture.

## Apps

- `client/`: React + Vite + TypeScript + TailwindCSS v4 + Redux Toolkit
- `server/`: Express + TypeScript + MongoDB + JWT auth via httpOnly cookies

## Phase 1

- Project setup
- Design system baseline
- Auth layout + main layout
- Register / login / logout
- Protected route foundation

## Run

```bash
npm run dev
```
